Help

This is a tool to clean up and organise notes to make studying efficient for students

Main features:

  • Upload pictures of notes and use a machine learning algorithm to transfer it to text
  • Upload slides and take notes and organise them by slide
  • Plain notes are your thing? Don't worry, we have you covered! Start a blank note

How to use it:

  • Upload your content to your notebook using the "New Note" padlet
  • Select how you would like to write your notes and write them out
  • Save them to dashboard to be studied later on/edited later on
